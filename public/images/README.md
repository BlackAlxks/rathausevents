# Image Assets

This folder contains all image assets for the Rathaus Friedrichshagen website.

## Folder Structure

- **`gallery/`** - Images for the main gallery page showcasing events and venue photos
- **`hero/`** - Hero/banner images for the homepage and section headers
- **`venue/`** - Photos of the Rathaus building, rooms, and facilities
- **`events/`** - Photos from past events and celebrations
- **`logos/`** - Rathaus Friedrichshagen logos and branding elements
- **`team/`** - Team member photos (if needed)

## Image Guidelines

- Use high-quality images (recommended: 1920x1080 or higher for hero images)
- Optimize images before uploading (use WebP format when possible)
- Use descriptive filenames (e.g., `rathaus-exterior-winter.jpg`)
- Keep image file sizes reasonable for web performance (< 500KB per image)
